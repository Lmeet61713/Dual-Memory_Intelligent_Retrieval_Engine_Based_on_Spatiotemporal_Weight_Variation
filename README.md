# 🏠 Homer – 具身智能家庭物品记忆助手

> 看见房间，记住物品，在对话中成长——回答你的每一个“找东西”的问题。

Homer 是一个集**双摄像头目标检测**、**动态物品显著性**、**空间与语义记忆**、**对话反馈自适应学习**与 **LLM Agent** 于一体的具身智能家庭物品管理助手。  
它能实时观察房间，记住每样东西的位置和“活跃度”，并且在每一次自然对话中主动修正和强化自己的记忆，最终像一个真正了解这个家的伙伴一样帮你快速找到物品。

---

## ✨ 核心创新

- **双摄像头交叉验证**  
  全局俯视 + 移动视角同步检测，只有两个摄像头“同时看到”的物品才会被确认记录，大幅降低误检测。

- **动态物品显著性（Object Dynamic Salience）**  
  物品不是平等的——越常被移动的东西越“重要”，越容易被优先找回。  
  系统通过“移动奖励 + 静止衰减 + 消失衰减”三线机制，模拟人类的记忆规律，让记忆随时间自然演化。

- **空间记忆 + 物品关系图谱**  
  不只是记录物品的坐标和周围参照物，我们构建了一个**可学习的物品关系图**。从摄像头同帧共现和对话反馈中自动提取物品间的功能关联（如“菜刀–砧板”、“水杯–药瓶”），使系统具备联想推理能力。当用户说不清物品名字或物品暂时被遮挡时，图谱可帮助系统给出有依据的推测。

- **语义记忆（对话话题管理）**  
  对话会自动分段，每个话题由大模型生成摘要并保存为向量。当用户聊到之前的事情时，Homer 能主动回忆起相关上下文，真正实现长期对话记忆。

- **LLM Agent 自主决策**  
  基于 LangChain 的智能体会自主判断何时查询物品位置、何时搜索对话历史，无需固定的唤醒词。你只需要像跟人说话一样，说“我的水杯在哪儿？”“还记得上次我们聊的那本书吗？”

- **对话反馈自适应学习（Human-in-the-Loop）**  
  系统不仅从摄像头学习，**更从用户的对话反馈中学习**。当用户纠正物品位置（“不在茶几，在书桌上”），或说出物品间的隐含关系（“洗洁精我放洗衣机旁边了”），系统会自动修正空间记忆、更新关系图权重，并调整物品的置信度——真正做到越用越聪明。

- **动态用户画像**  
  通过对话内容持续提取用户的性格特质、兴趣偏好、生活习惯，并在最终回复中融入这些信息，使每一次交互都更贴心、更个性化。

- **流式前端交互**  
  FastAPI + SSE 实现的流式聊天，支持历史会话管理，摄像头一键启停。

## 🚀 快速开始

### 1. 环境准备

- Python 3.10+
- 两个 USB 摄像头（或内置摄像头 + 外接，也可仅用对话模式）
- 建议使用虚拟环境

```bash
git clone https://github.com/你的用户名/homer.git
cd homer
pip install -r requirements.txt
```

### 2. 配置

在项目根目录创建 `.env` 文件，填入必要 API Key（或设置系统环境变量）：

```
DEEPSEEK_API_KEY=sk-xxxxxxxx
```

编辑 `config.py` 中与硬件相关的参数（若无需摄像头，可跳过）：

```python
GLOBAL_CAM_ID = 0    # 全局摄像头设备索引
MOBILE_CAM_ID = 1    # 移动摄像头设备索引
```

### 3. 启动

```bash
python main.py
```

打开浏览器访问 `http://localhost:8501`，在侧边栏点击“开启摄像头”启动物体检测，然后就可以在聊天框里向 Homer 提问了。  
即使不开启摄像头，系统也能通过纯对话模式记忆物品和进行推理。

## 🔬 关键技术剖析

### 物品关系图（Object Relationship Graph）
- **数据来源**：双摄同帧共现、对话中用户陈述的物品关联（“放在...旁边”、“和...一起用”）。
- **更新机制**：每次观测或对话反馈时，对应物品间的边权重增加；长期未被提及的关联会随时间缓慢衰减。
- **推理应用**：当用户问题缺少明确物品名时，系统通过图游走寻找最可能的关联物品，实现“联想检索”。

### 对话反馈自适应
- **意图识别**：在每次 Agent 回复后，系统解析用户下一句话中是否包含对上次回答的纠正、补充或确认。
- **空间修正**：若用户指出物品正确位置，系统自动更新坐标并降低旧参照物权重。
- **关系学习**：若出现“A 在 B 旁边”等句式，自动强化图关系边。
- **画像更新**：从日常对话中提取长期稳定的性格特质与偏好，融入回复生成。

### 动态显著性模型
- 新物品初始权重 0.5，最大 0.99，最小 0.20。
- 双摄观察到物品移动 → 奖励 +0.1。
- 同一位置连续检测到但未移动 → 步长累计，达到阈值衰减 -0.05。
- 超过一定时间未被任何摄像头观测 → 消失衰减 -0.05（可调整）。
- **对话反馈可额外触发奖励/惩罚**：被用户确认过位置的物品获得小幅奖励；被频繁纠正的物品降低空间可信度。

## 🔭 未来计划（Roadmap）

- [ ] **物品关系图可视化**：在前端展示物品之间的关联网络。
- [ ] **更细粒度的对话反馈解析**：利用更强模型提取用户隐含的意图和情绪。
- [ ] **视觉特征融合**：引入 CLIP 等模型，直接用图像特征强化空间记忆。
- [ ] **多空间/多用户支持**：为不同房间和家庭成员维护独立的记忆和图谱。
- [ ] **主动推送**：当某物品显著性异常升高但用户长时间未询问时，主动提醒。
- [ ] **开放 API**：提供标准化 REST API，便于集成到智能音箱、机器人等平台。

## 🧪 学术与商业价值

- **学术**：Homer 在具身智能中率先引入了**基于对话反馈的自适应记忆修正**和**可学习物品关系图**，探索了神经-符号系统在长期家庭交互场景下的应用，为“持续学习的服务机器人”提供了新范式。
- **商业**：面向健忘人群、老年人、共享居住空间等，提供一种越用越懂你的物品管理方案。其核心算法可被轻易移植到各类家用机器人或智能音箱中，具备清晰的商业化路径。
